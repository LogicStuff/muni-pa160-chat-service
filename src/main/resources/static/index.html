PA160 Chat Web Service

This is an example of a web service. It is implemented using the Spring Boot framework in Java. Its sources are available at https://github.com/martin-kuba/muni-pa160-chat-service, its implementation is in ChatService.java.

Visit /chat with your browser to see all messages on an HTML page.

Visit /api/messages with your browser to get all messages in JSON format.

Post a new message using command line (replace the URL prefix with real value):

    curl -X POST 'https://ip-78-128-251-137.flt.cloud.muni.cz:8443/api/messages?text=Hello+World+!'

Open the OpenAPI description in Swagger Editor , and try to call the service.

Bonus: Python client

In the Swagger Editor, click Generate Client - python in the top menu. A file named python-client-generated.zip will be downloaded. Extract the ZIP file into an empty folder. Read the extracted README.md file, namely the part titled „Getting Started”. Copy the example client code into a new file in the same folder, and add the following two lines just after the imports: 
configuration = swagger_client.configuration.Configuration()
configuration.host = 'https://ip-78-128-251-137.flt.cloud.muni.cz:8443/api'
where you need to replace localhost with the actual name of the server.

Run the python client.
