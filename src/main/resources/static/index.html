PA160 Chat Web Service

This is an example of a web service. It is implemented using the Spring Boot framework in Java. Its sources are available at https://github.com/martin-kuba/muni-pa160-chat-service.

Visit /chat with your browser to see all messages on an HTML page.

Visit /api/messages with your browser to get all messages in JSON format.

Post a new message using command line (replace localhost with the DNS name of the server):

    curl -X POST 'http://localhost:8080/api/messages?text=Hello+World+!'

Open the OpenAPI description in Swagger Editor
